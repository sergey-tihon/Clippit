Remarks

A worksheet can include ColumnHeadings and a TableName. When both are provided and at least one data row exists, a formal Excel table definition is created with auto-filter and "TableStyleMedium2" styling. Sheet names must be valid XLSX names (max 31 characters, no special characters like * [ ] / \ : ?).
