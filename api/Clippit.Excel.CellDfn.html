Remarks

Value accepts the following CLR types:

  • string
  • bool
  • DateTime and DateTimeOffset
  • Numeric types: int, long, uint, double, float, etc.

Use FormatCode with a standard Excel format string (see StandardFormats) or a custom format string to control how the value is displayed.
