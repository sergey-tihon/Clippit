## Build Instructions

### Build

Run `./build.sh` on Unix systems or `.\build.cmd` on Windows.

### Documentation

- Install DocFx

  - MacOS: `brew install docfx`
  - Windows : `choco install docfx -y`

- Run `docfx docs/docfx.json --serve` to start local copy of site/docs.
